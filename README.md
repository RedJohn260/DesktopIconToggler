# Desktop Icon Toggler 🖥️✨

![Banner](Images/Banner.png)

A lightweight, high-performance Windows tray application to toggle desktop icons with a hotkey.

### 🚀 Features
* **Global Hotkey:** Toggle icons instantly (Default: `Ctrl + D`).
* **Ultra Lightweight:** Uses only ~20MB of RAM.
* **Startup Support:** Option to run when Windows starts.
* **Portable:** Single `.exe` file, no installation required.

### 🛠️ Installation
1. Go to the **Releases** tab on the right.
2. Download `DesktopIconToggler.exe`.
3. Run it and find it in your system tray!

### 💻 Development
Built with C# and .NET 10.0 using Visual Studio 2026.
